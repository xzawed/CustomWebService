# 무료 API 카탈로그 (Free API Catalog)

## 선정 기준
1. **영구 무료**: 무료 플랜이 별도 결제 없이 영구적으로 제공
2. **안정성**: 최소 2년 이상 운영된 서비스
3. **문서화**: API 문서가 잘 정리되어 있음
4. **인증 간편**: API 키 발급이 무료이고 간단함
5. **호출 제한**: 무료 한도가 소규모 서비스에 충분

## API 카탈로그 관리 정책

### 추가 절차
1. API가 위 선정 기준을 충족하는지 검증
2. 무료 정책 변경 이력 확인
3. 테스트 호출로 정상 동작 확인
4. 카탈로그에 추가 및 연동 어댑터 개발

### 제거 절차
1. API 서비스 종료 또는 무료 정책 변경 시
2. 해당 API를 사용 중인 서비스 사용자에게 알림
3. 대체 API 제안
4. 유예 기간(30일) 후 카탈로그에서 제거

### 모니터링
- 주간: 각 API 정상 동작 여부 자동 체크
- 월간: 무료 정책 변경 여부 수동 확인
- 수시: 커뮤니티 리포트 반영
