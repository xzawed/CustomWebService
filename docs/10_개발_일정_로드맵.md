# 개발 일정 및 로드맵 (Development Roadmap)

## 마일스톤 개요

```
Phase 1: MVP 기반 구축
  → 프로젝트 셋업, DB, 인증, API 카탈로그

Phase 2: 핵심 기능 개발
  → 빌더 UI, AI 코드 생성, 미리보기

Phase 3: 배포 & 관리
  → 자동 배포, 대시보드

Phase 4: 안정화 & 출시
  → 테스트, 최적화, 베타 출시
```
