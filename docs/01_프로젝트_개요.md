# 프로젝트 개요 (Project Overview)

## 프로젝트명
**CustomWebService** - 무료 API 기반 맞춤형 웹서비스 자동 생성 플랫폼

## 프로젝트 비전
비개발자도 원하는 무료 API를 선택하고 서비스 컨텍스트를 입력하면, AI가 자동으로 웹서비스를 생성하여 배포까지 완료해주는 올인원 플랫폼

## 핵심 가치
- **무료**: 모든 API, 호스팅, 운영 비용 제로
- **쉬움**: 드래그 앤 드롭 / 체크박스 기반 비개발자 친화 UI
- **자동화**: 컨텍스트 입력만으로 웹서비스 자동 생성
- **즉시 사용**: 생성 즉시 배포되어 바로 사용 가능

## 서비스 흐름 요약

```
[사용자] → [API 카탈로그에서 API 선택] → [서비스 컨텍스트 입력] → [AI 웹서비스 생성] → [자동 배포] → [사용자에게 URL 제공]
```

### 단계별 설명

1. **API 카탈로그 탐색**: 카테고리별로 정리된 무료 API 목록을 탐색
2. **API 선택**: 드래그 앤 드롭 또는 체크박스로 원하는 API를 다중 선택
3. **컨텍스트 입력**: 어떤 서비스를 만들고 싶은지 자연어로 설명
4. **웹서비스 생성**: AI가 선택된 API와 컨텍스트를 기반으로 웹서비스 코드 생성
5. **자동 배포**: 생성된 웹서비스를 무료 호스팅에 자동 배포
6. **결과 전달**: 생성된 웹서비스 URL과 관리 페이지 제공

## 대상 사용자
- 아이디어는 있지만 개발 능력이 없는 비개발자
- 빠르게 프로토타입을 만들고 싶은 기획자
- 무료 API를 활용한 사이드 프로젝트를 원하는 사용자
- 소규모 비즈니스에 간단한 웹서비스가 필요한 사업자

## 프로젝트 범위

### In Scope
- 무료 API 카탈로그 관리 (영구 무료 API만 수록)
- 드래그 앤 드롭 / 체크박스 기반 API 선택 UI
- 자연어 컨텍스트 입력 및 처리
- AI 기반 웹서비스 코드 자동 생성
- 무료 호스팅 자동 배포 (Railway / GitHub Pages)
- 생성된 서비스 관리 대시보드

### Out of Scope (v1 기준)
- 유료 API 연동
- 커스텀 도메인 연결
- 데이터베이스가 필요한 복잡한 서비스
- 사용자 인증/로그인이 필요한 서비스
- 모바일 앱 생성
