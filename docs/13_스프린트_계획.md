# 스프린트 계획 (Sprint Plan)

## 스프린트 운영 원칙
- **스프린트 주기**: 1주 단위
- **총 스프린트**: 8 스프린트
- **각 스프린트 구조**: 기능 개발 → 테스트 → 코드 리뷰 → 회고
