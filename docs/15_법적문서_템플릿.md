## 1. 이용약관 (Terms of Service)

### 게시 위치: `/terms`

```
CustomWebService 이용약관

최종 수정일: [날짜]

제1조 (목적)
본 약관은 CustomWebService(이하 "서비스")의 이용 조건 및 절차,
이용자와 서비스 제공자 간의 권리·의무를 규정합니다.

제2조 (정의)
1. "서비스"란 사용자가 무료 API를 선택하고 서비스 설명을 입력하면
   AI가 웹서비스를 자동 생성·배포하는 플랫폼을 의미합니다.
2. "사용자"란 본 약관에 동의하고 서비스를 이용하는 자를 의미합니다.
3. "생성 서비스"란 사용자의 요청에 따라 AI가 자동 생성한 웹서비스를 의미합니다.

제3조 (서비스 제공)
1. 서비스는 무료로 제공됩니다.
2. 서비스 제공자는 사전 고지 없이 서비스의 내용을 변경하거나
   중단할 수 있습니다.
3. 서비스는 무료 외부 API에 의존하므로, 해당 API의 정책 변경에 따라
   서비스 기능이 제한될 수 있습니다.

제4조 (사용자 의무)
1. 사용자는 다음 행위를 해서는 안 됩니다:
   - 불법적인 목적의 서비스 생성
   - 타인의 권리를 침해하는 서비스 생성
   - 악성 코드를 포함하는 서비스 생성 시도
   - 서비스의 정상 운영을 방해하는 행위
   - API의 이용 정책을 위반하는 행위
2. 사용자는 생성된 서비스의 내용에 대해 책임을 집니다.

제5조 (지적 재산권)
1. 서비스 플랫폼의 지적 재산권은 서비스 제공자에게 있습니다.
2. AI가 생성한 코드의 저작권은 사용자에게 귀속됩니다.
3. 사용된 무료 API의 데이터는 각 API 제공자의 라이선스를 따릅니다.

제6조 (면책 조항)
1. AI가 생성한 코드의 정확성, 안전성, 완전성을 보장하지 않습니다.
2. 생성된 서비스로 인해 발생한 손해에 대해 책임지지 않습니다.
3. 무료 API의 장애, 정책 변경으로 인한 서비스 중단에 대해
   책임지지 않습니다.
4. 무료 호스팅 플랫폼의 장애로 인한 서비스 중단에 대해
   책임지지 않습니다.

제7조 (서비스 이용 제한)
1. 사용자당 일일 서비스 생성 횟수: 10회
2. 프로젝트당 최대 API 선택 수: 5개
3. 위 제한은 서비스 운영 상황에 따라 변경될 수 있습니다.

제8조 (계정 및 데이터)
1. 사용자는 언제든지 계정을 삭제할 수 있습니다.
2. 계정 삭제 시 관련 데이터(프로젝트, 생성 코드)는 삭제됩니다.
3. 배포된 서비스는 별도로 삭제 요청해야 합니다.

제9조 (약관 변경)
1. 약관 변경 시 서비스 내 공지합니다.
2. 변경된 약관에 동의하지 않는 사용자는 서비스 이용을 중단할 수 있습니다.

제10조 (준거법 및 관할)
본 약관은 대한민국 법을 준거법으로 합니다.
```

## 2. 개인정보처리방침 (Privacy Policy)

### 게시 위치: `/privacy`

```
CustomWebService 개인정보처리방침

최종 수정일: [날짜]

1. 수집하는 개인정보

가. 필수 수집 항목
- 소셜 로그인 시: 이메일 주소, 이름, 프로필 이미지 URL
  (Google/GitHub 제공 정보)

나. 자동 수집 항목
- 서비스 이용 기록 (생성한 프로젝트, 선택한 API)
- 접속 로그 (IP 주소, 브라우저 정보, 접속 시간)

다. 수집하지 않는 항목
- 결제 정보 (무료 서비스)
- 주민등록번호, 전화번호
- 위치 정보

2. 개인정보의 이용 목적

- 서비스 제공 및 사용자 인증
- 생성된 프로젝트 관리
- 서비스 개선 및 통계 분석
- 장애 대응 및 기술 지원

3. 개인정보의 보관 및 파기

가. 보관 기간
- 계정 정보: 회원 탈퇴 시까지
- 서비스 이용 기록: 마지막 이용일로부터 1년
- 접속 로그: 3개월

나. 파기 절차
- 보관 기간 경과 시 자동 삭제
- 회원 탈퇴 요청 시 즉시 삭제 (배포된 서비스 포함)

개인정보를 제3자에게 제공하지 않습니다.
단, 다음의 경우 예외입니다:
- 법령에 따른 요청이 있는 경우
- 사용자가 사전에 동의한 경우

5. 개인정보의 처리 위탁

가. 위탁 현황
| 수탁자 | 위탁 업무 | 보관 위치 |
|--------|-----------|----------|
| Supabase | 데이터베이스 호스팅 | AWS (ap-northeast-1) |
| Vercel | 웹 호스팅 | AWS |
| Google | 소셜 로그인 인증 | Google Cloud |
| GitHub | 소셜 로그인 인증, 코드 저장 | GitHub |

6. 사용자의 권리

사용자는 다음 권리를 행사할 수 있습니다:
- 개인정보 열람 요청
- 개인정보 수정 요청
- 개인정보 삭제 요청 (계정 삭제)
- 개인정보 처리 정지 요청

7. 쿠키 사용

가. 사용 목적
- 로그인 세션 유지 (필수 쿠키)

나. 사용하지 않는 쿠키
- 광고 추적 쿠키
- 제3자 마케팅 쿠키

8. 개인정보 보호 조치

- 데이터 전송 시 HTTPS(TLS) 암호화
- 데이터베이스 Row Level Security 적용
- API 키 암호화 저장
- 접근 권한 최소화

9. 개인정보 보호 책임자

- 이메일: [이메일 주소]

10. 방침 변경

본 방침 변경 시 서비스 내 공지합니다.
```

## 4. 면책 조항 (Disclaimer)

### 게시 위치: 서비스 생성 완료 시 안내 + `/disclaimer`

```
면책 조항

1. AI 생성 코드 관련
   - AI가 생성한 코드는 참고 목적이며, 프로덕션 환경에서의
     사용 시 사용자의 책임하에 검토 후 사용해야 합니다.
   - 생성된 코드의 보안 취약점, 버그, 성능 이슈에 대해
     서비스 제공자는 책임지지 않습니다.

2. 무료 API 관련
   - 사용된 API는 각 제공자의 정책에 따라 중단, 변경,
     유료 전환될 수 있습니다.
   - API 장애 또는 데이터 오류로 인한 손해에 대해
     서비스 제공자는 책임지지 않습니다.

3. 호스팅 관련
   - 무료 호스팅 플랫폼의 정책 변경 또는 장애로 인해
     배포된 서비스가 중단될 수 있습니다.
   - 서비스의 가용성을 보장하지 않습니다.

4. 데이터 관련
   - 무료 서비스 특성상 데이터 백업을 보장하지 않습니다.
   - 중요한 데이터는 사용자가 별도 백업해야 합니다.
```
